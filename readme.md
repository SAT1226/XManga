# XManga
 XManga is simple comic viewer written C++ and Qt.

## Feature
* Lanczos3 resize filter (ImageMagick's Lanczos3 implementation)
* Spread viewing
* Magnifying Lens
* Key customize
* Bookmark

## Dependencies
* Qt5
* [unarr](https://github.com/selmf/unarr)

## Installation
```
$ cmake .
$ make
$ sudo make install
```
